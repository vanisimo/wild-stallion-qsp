# Design: мэр, печать, коррупция семьи (черновик сюжета)

**Статус:** дизайн согласован по блокам А–Ж (сессия 2026-06), **в QSP не реализовано**.  
**Связь с кодом:** этап 1 мэрии и отработка — `docs/state.md`; свидетельство/дворянин — частично в коде, нужна переделка под этот документ.  
**Следующий шаг:** playtest текущего кода → затем реализация по разделам ниже.

## Принципы UI и свободы игрока

- **Нет кнопок вилки** («честно / грязно / шантаж») — только блоки **мыслей Стефана** (плюсы, минусы, риски).
- **Диалоги** — через **меню по образцу girl talk** (темы, без принудительного «Закрыть»).
- Игрок **не загнан в рамки**: можно вести доходный трактир, собирать слухи, идти честным путём **без** отдачи сестёр мэру.
- **Мэр не забирает** сестёр силой — только **озвучивает желание**; визит — только если игрок сам уговорил **всех троих**.
- **Мэр не торопит** — торопит только **желание ГГ стать дворянином**.
- **Ветки параллельны**: честный путь, дом, слабый/сильный шантаж, порт — одновременно.

## Любовница мэра = Ирма (не Жоржетта)

- Слух **общедоступный, рано** — клерк, зал, пятница, Бекки; **жена мэра знает, всем плевать**.
- При **хороших отношениях** Ирма может **сама** обмолвиться.
- Доп. входы: подслушивание **Сандра + Бекки** (воскресенье), **исповедь** (полдень воскресенья).
- Флаг: `PlayerKnows_MayorIrmaAffair` (ранний, «все знают»).

**Слабый шантаж** (Ирма): мэр **смеётся** — *«Наивный мальчик… полгорода… и маму твою тоже»*. Любовные связи не пугают.

**Мысли ГГ:** нужен компромат **убойнее** — порт, **поставки оружия** (TBD, сложно узнать).

## Первый приём по бумаге (после отработки клерку)

**Уже в коде:** отработка → `MayorFirstTalkDone = 1` (базовая сцена).

**Планируемая доработка сцены** (не площадь, только кабинет):

1. Гейт: **приличный костюм** + отработка.
2. Мэр **сразу озвучивает два пути** (официальный голос + короче второй):

   **Путь 1 — честный** (условия **ставит мэр**, не скрытый чеклист):
   - ~**21 день** безупречной работы трактира после этого разговора;
   - стабильные **налоги в казну** (черновик: +900 поверх уже уплаченного);
   - **репутация** без пятен;
   - запрос в **столицу**, **бархатная книга**, деньги, время;
   - **печати сейчас нет**.

   **Путь 2 — ускорение** (другой тон, слащавее, без пафоса):
   - если репутация **не будет кристальной** — **зайдите снова**, мэр **обозначит условия** (на втором приёме).

3. **Всегда:** дворянский костюм у **Ирмы**, очень дорого.

4. Блок **мыслей Стефана** (три направления + плюсы/минусы):
   - **Честно** — долго, дорого, семья целее;
   - **Коррупция** — быстрее, цена семье и репутации;
   - **Шантаж** — нужен сильный компромат; Ирма слаба; порт/оружие впереди.

5. **Без выбора вилки** — игрок уходит и действует сам.

## Второй приём (опциональный)

**Открытие (черновик):** падение `tavern_reputation` + высокая **семейная коррупция** (`FamilyCorruptionStage` / `FamilyTension` — пороги уточнить после playtest).

Мэр **не ультиматум** — **желание**: сёстры; намёк, что **маму уже «пользует»** (foreshadow обязателен).

Игрок может уйти с **мыслями**: шантаж / уговорить дома / продолжать честный путь.

## Мама (Сандра): путь к согласию

### Foreshadow

- **Драупнир (гном):** при визите всегда про мать **с теплотой** — «хорошая женщина».
- **Драупнир знает** про мэра-ходока; **принимает**; **рад**, что у него есть Сандра.
- **Сундук Сандры:** фалоимитатор из **спецдерева**, на ощупь «как живой».
- **Вопрос:** краснеет — сделал **«очень хороший человек»** (гном-столяр).
- **Воскресенье, лавка продовольствия:** подслушивание — слух **Сандра + гном**.

Флаг: `PlayerKnows_SandraDraupnirAffair` (сундук и/или подслушивание).

### Разговор поддержки

Когда знаем про гнома (или узнали раньше):

- ГГ **не осуждает** связь с **гномом**;
- **мэр** — «необходимость», тоже без морали.

→ Сандра **согласна** на схему с мэром (не радость — согласие).

Без знания про гнома — **блок** / отказ обсуждать.

Foreshadow **мэр + мама:** Бекки, исповедь, усталость Сандры после города, реплика мэра на слабом шантаже.

## Визит к мэру (сёстры)

**Условие:** согласны **все трое** (Аманда, Мелисса, Сандра).

1. Игрок идёт к мэру → мэр **назначает время** (не в тот же день).
2. Девушки идут **без ГГ**.
3. После — **отдельный разговор с каждой** (меню girl talk).

### Реакции после визита

| Кто | Реакция |
|-----|---------|
| **Мелисса** | **Жалела**; извращенец, не понравилось, терпела; **больше не хочу**, даже с украшением; может **ужесточить контроль** в зале |
| **Аманда** | «За такой подарок готова хоть с **кентавром**»; ей **норм** |
| **Сандра** | **Молчит** в общем; в личном — смущение |

**Подробности** визита к мэру — при **большом otkroven** у каждой (не чеклист, доверительный talk).

## Честный путь (параллельно)

Условия озвучены на **первом приёме**. Игрок может:

- копить 21 день + налоги + репутацию;

Визит сестёр **не блокирует** честный путь механически; **FamilyCorruption** влияет на «чистую» концовку позже.

## Арка бумаги и дворянина (согласовано, блоки А–Б)

### Сундук Сандры (до дворянина)

- **Условие:** `day > 7`, Сандры **нет в комнате** (работает на кухне), не ночь.
- **Осмотр 1:** письмо **Лермонта** — любовь, сожаление об отъезде «навсегда».
- **Разговор с мамой** (girl talk, **без порогов**): застигнута врасплох, «дела далёких дней», без конкретики. Мысли ГГ — отдельный блок.
- **Осмотр 2 (позже):** дилдо при высоком `sluttiness['sandra']` — она меньше прячет → намёк на гнома.
- Письмо можно найти **позже** (после дворянина), но **без влияния на сюжет** — только лор.
- Доп. знание **Сандра+гном:** под окнами лавки Бекки (воскресенье), сундук, реплики Драупнира — **предвестники**, можно найти все или ни одного.

### Дворянин в зале

- **Приоритетный** ивент, **без привязки** к сундуку Сандры.
- Дворянин **абстрактный** (не Лермонт): увидел, что ГГ не поклонился → конфликт.

### После дворянина (всегда)

- Сандра в разговоре **сама** говорит про тайное дно **сундука Стефана**; Стефан злится, она успокаивает.
- Свидетельство в сундуке ГГ — **всегда**, независимо от сундука Сандры.
- Отец в записи — дворянин; связь с письмом Лермонта — лор, не имя в зале.

- Низкая семейная коррупция — возмущение.
- Высокая — грубее: «хочет моих сестёр… козлина», но «с них не убудет».

## Церковь, мэр, пятница после 2-го визита (блок Д — предложение агента)

### Индекс давления «грязный трактир» `TavernScandalPressure` (0–100)

Считается раз в неделю (воскресное утро) или при триггере:

| Фактор | Вклад |
|--------|--------|
| `tavern_sex_appeal >= 40` | +15 |
| Откровенная униформа **включена у всех троих** и хоть одна в смене | +20 |
| `FamilyCorruptionStage >= 2` | +10 за ступень |
| Провал инспекции стражи за 7 дней | +15 |
| `CityRumorLevel` от слухов о трактире | +5 за слух |
| Спонсорство пятницы **на этой неделе** | **−30** |
| Пожертвование церкви **≥50** на этой неделе | **−25** |
| `tavern_reputation >= 60` | −10 |

**Церковь ругает** (воскресная служба/проповедь): `MayorSecondTalkDone = 1` **и** `TavernScandalPressure >= 45` **и** нет пожертвования ≥50 на неделе.

**Мэр ругает** (не в кабинете — в городском контексте / клерк): те же условия **и** `TavernCityTaxTotal` за последние 7 дней < 150 **и** нет спонсорства.

**Мэр хвалит в пятницу** (речь перед танцами): `MayorSecondTalkDone = 1` **и** (`TavernCityTaxPaidWeek >= 200` **или** `FridayDanceSponsoredWeek = 1`). Текст хвалы **снижает** эффект ругани (−20 к perceived scandal на неделю).

Флаги: `TavernScandalPressure`, `FridayDanceSponsoredWeek`, `ChurchDonationThisWeek`, `MayorSecondTalkDone`.

## Система слухов (блок Е — согласовано + предложение)

### Правила

- **Ирма+мэр:** при **первом** использовании источника (рынок / пятница / порт) — **100%**, без привязки к дню.
- **Недавнее событие** (≤7 дней): шанс ×**1.75** (округление вверх).
- **Сандра+гном (порт):** слух раз в неделю, если **Сандры нет дома ночью** (после этого флаг слуха доступен).
- **Мэр+мама:** не слух — реплика мэра при слабом шантаже (см. визит 2).

### Таблица источников

| Источник | Когда | Базовый шанс | Первый контакт | Пул слухов |
|----------|--------|--------------|----------------|------------|
| Прогулка рынок | день, −1 слот | 15% | Ирма+мэр 100% | город, торговцы |
| Пятница, между людьми | вечер | 35% | Ирма+мэр 100% | мэр, танцы |
| Воскресенье, разговор с семьёй на службе | утро, 1× | 30% | — | общие |
| Исповедь, подслушать | полдень | 55% | — | грехи, церковь |
| Под окно лавки Бекки | вс **день/вечер** | 30% | — | Сандра+Бекки, дети, **Сандра+гном** |
| Под окно винной | вс день/вечер | 30% | — | Кларисса+Мелисса |
| 2-й этаж таверны | вечер | 35% | — | Аманда+Лизетта (тексты из старой игры) |
| Порт, прогулка | день, −1 слот | 15% | Ирма+мэр 100% | порт; Сандра+гном с ~10 дн. |

### Подслушивание в воскресенье (предложение агента)

- Доступно: **воскресенье**, `time = 3` (день) и `time = 4` (вечер).
- В каждой локации с окном — кнопка **«Подслушать»**.
- **1 попытка на локацию на слот**; если выпал «пусто» — **вторая попытка в том же слоте и локации** с шансом ×0.5.
- Успешное подслушивание **не** двигает время; **вторая локация в том же слоте** — можно; переход день→вечер — обычным перемещением по городу.
- Альтернатива (если слишком щедро): одна кнопка «Погулять и подслушать» за слот — выбор **одной** локации, затем `time += 1`.

Рекомендация: **вариант 1** (2 попытки в одной локации за слот, время не тратится на слушание) — больше контроля игроку.

Knowledge: `PlayerKnows_MayorIrmaAffair`, `PlayerKnows_SandraDraupnirAffair`, …

## Откровенная униформа (согласовано)

**Принцип:** в зале включается **только у всех троих сразу** (один общий переключатель). Покупка и сцены — **по одной девушке**.

### Покупка (на девушку)

1. Тема у Ирмы открыта (скандал / репутация / церковь — как в текущем `IrmaUniformOffer`).
2. **Высокая личная коррупция** девушки (`sluttiness[girl]` — порог TBD после playtest).
3. **Отдельный разговор** с девушкой (girl talk) — согласие на заказ.
4. **На следующий день** — визит к Ирме, примерка:
   - подглядывание за переодеванием;
   - приставание к девушке;
   - приставание к портнихе при **высоких** `Friends['irma']`.
5. `uniform_purchased[girl] = 1`.

### Ношение в смене (все трое)

Переключатель «откровенная форма в зале» доступен только если:

- `uniform_purchased['amanda'] = 1` **и** `melissa` **и** `sandra`;
- у **каждой** есть сшитое **платье/комплект** (куплено у Ирмы);
- ГГ **обсудил переход** на новую форму с **каждой** (`GirlUniformTransitionTalkDone[girl] = 1`).

Иначе девушки остаются в обычной форме, даже если комплект куплен.

### Влияние на `TavernScandalPressure`

+20 только когда переключатель **включён** и хотя бы одна в смене в откровенной форме.

Флаги: `RevealingUniformHallEnabled`, `GirlUniformTransitionTalkDone[girl]`, `GirlUniformFittingDay[girl]`.

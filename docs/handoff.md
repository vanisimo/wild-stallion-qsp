## 1. Что делали в этой сессии (обсуждение, без нового кода)

### 1.1. Сверка с story bible

- Источник: `wild_stallion_story_bible_v4.docx` (v4, русский).
- Главная идея bible: трактир — **точка влияния на город**, не самоцель; мэр-прагматик помогает, если трактир «кормит» город.
- В bible **нет числовой экономики** (мараведи, налоги, доли) — только сюжетные связи.

### 1.2. Переделка доступа к мэру (этап 1 — утверждённый план, НЕ в коде)

**Проблема текущего кода:** `MayorOfficeMagistrateSubmit` сразу берёт «госпошлину» 100–145 мараведи и запускает магистрат. Это **обходит** задуманный гейт через экономику.

**Целевой flow этапа 1 (путь «развитие трактира»):**

1. Свидетельство в тайном отделении сундука (`BirthCertificateFound`).
2. Клерк **отказывает** / отсылает к мэру без приёма.
3. Игрок качает доходы таверны → растёт **налог 10%** городу (`TaxPercent` уже есть в `business_schedule.qsps`, **списание не реализовано**).
4. Накопить `TavernCityTaxTotal >= 600` мараведи уплаченного налога.
5. Взятка за приём у мэра **~180** мараведи.
6. Первая сцена с мэром → `MayorFirstTalkDone`.
7. **Без** `StefanNobleLineUnlocked` на этом этапе — подтверждение рода **после** приёма у мэра.

### 1.3. Полный аудит экономики (анализ кода)

Сводка для баланса — **что есть / чего нет**.

#### Старт (`game_init.qsps`)

- `money = 800`
- Цены: еда 15, вино 20, пиво 10
- Запасы: продукты 300, вино/пиво 120
- `tavernvisitors = 25`, `tavern_reputation = 40`
- Работы девушек **все 0** — без назначения персонала дохода нет

#### Закупки

| Поставщик | Товар | Закуп | Продажа | Маржа |
|-----------|-------|-------|---------|-------|
| Бекки | продукты | 4 м/ед (ящики 60–500) | 15 | ~73% |
| Легаре | вино | 8.8 м/ед (220–1550) | 20 | ~56% |
| Легаре | пиво | 3.75 м/ед (150–700) | 10 | ~62% |

#### Типичный день (Сандра кухня / Мелисса зал / Аманда уборка, по 1 работе)

| Статья | ~мараведи |
|--------|-----------|
| Валовый доход | **341** |
| «Расходы персонала» (12×3) | −36 |
| Чистая прибыль | **~305** |
| Закупка запасов (реальный COGS, не авто-списание) | **~190** |
| Налог 10% (план, не в коде) | −34 |
| Свободный остаток после налога | **~81/день** |

- Работа за стойкой: **+15–40** сразу в `money` (`tavern_bar_work.qsps`).
- Пиво расходуется **в 2× быстрее вина** — узкое место (~8 дней со стартовых 120 ед.).

#### «ЗП» в коде (`AddGirlWorkExpenses`)

- 1 работа: **12**/день на девушку
- 2 работы: **22**
- 3 работы: **42**

Это абстрактные расходы в формуле прибыли, **не** выплата совладелицам.

#### Доля девушек-совладелиц

**В коде отсутствует.** Вся прибыль → `money` ГГ.  
Предложение для дизайна: **20–40%** чистой прибыли → семейный фонд / `FamilyTension`.

#### Разовые расходы (уже в коде)

| Статья | Сумма |
|--------|-------|
| Ремонт Драупнира | 70–165 |
| Госпошлина магистрата | 100–145 |
| Взятка стражнику (Инга) | 300 |

| Униформа Ирмы (откровенная) | 80–135/дев, все трое ~270–320 |
| Подарки порта | 25–80 |
| Церковь | любая сумма, +1 rep |
| Штраф инспекции | −100…−200 к прибыли |

#### Чего нет в коде (нужно добавить)

| Статья | Предложенные цены |
|--------|-------------------|
| Платья для танцев (без них не ходят) | 60–90 у Ирмы |
| Одежда ГГ | 40–400 по статусу |

| Налог 10% | PR1 |
| Наёмный персонал | 8–15/день |
| Улучшения трактира (не ремонт) | 150–300 |

#### Ранний «пакет» крупных трат (~1600–1800)

Униформа + платья + ремонт + взятка Инге + накопить 600 налога + взятка мэру 180 + госпошлина → **2–3 недели** игры при текущем балансе.
